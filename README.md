  Desenvolvimento de uma calculadora de média utilizando JavaScript, HTML5 e CSS3.
  
  - Inserir 4 notas, somar e exibir a média
  - Para ser aprovado deve ter média 6
  - Exibe na tela a mensagem de aprovado ou reprovado
